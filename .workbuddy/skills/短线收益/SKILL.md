# 短线收益分析技能

## 概述

此技能用于自动化分析A股短线交易记录（v4.3），支持三种账户（两融/手机/平安），自动匹配买卖记录，计算含佣金（万一/双向/最低5元）和印花税（万五/卖出单边）的净盈亏，生成Excel汇总和HTML可视化报告，并自动提交到GitHub。

## 使用场景

当用户需要以下操作时，触发此技能：

- 分析股票交易记录
- 计算交易盈亏
- 生成投资报告
- 处理交易数据
- 查看短线收益情况
- 股票盈亏分析

## 核心功能

### 1. 自动化数据处理

- 自动识别Excel交易记录文件
- 智能匹配买卖记录（按股票代码配对）
- 加权平均计算成交均价
- 自动归档已处理文件

### 2. 智能计算逻辑（v4.3 含交易成本）

```
匹配数量 = min(买入数量, 卖出数量)
买入均价 = 总买入金额 / 总买入数量
卖出均价 = 总卖出金额 / 总卖出数量
毛盈亏 = 卖出金额 - 买入金额
佣金 = max(买入金额×0.01%, 5元) + max(卖出金额×0.01%, 5元)  — 万一/双向/最低5元
印花税 = 卖出金额 × 0.05%                                        — 万五/卖出单边
交易成本 = 佣金 + 印花税
净盈亏 = 毛盈亏 - 交易成本
盈亏比例 = 净盈亏 / 买入金额 × 100%
```

### 3. 数据去重机制

- 自动检测同一日期的重复数据
- 删除旧数据并替换为新数据
- 确保Excel汇总文件不重复

### 4. 报告生成

**单日HTML报告**：
- 文件路径：`reports/YYYY-MM-DD-股票交易盈亏报告.html`
- 内容：单日交易明细、盈亏统计

**汇总可视化报告**：
- 文件路径：`reports/汇总可视化报告.html`
- 内容：多维度图表分析
  - 总体统计卡片（交易天数、股票数、总盈亏等）
  - 每日盈亏柱状图
  - 累计收益曲线
  - 各股票盈亏分析（柱状图+饼图）

## 执行流程

### 步骤1：检查待处理文件

使用以下命令查找待处理的Excel文件：

```bash
python scripts/analyze_trading.py
```

或者直接在Python中导入：

```python
import sys
sys.path.insert(0, 'scripts目录路径')
from analyze_trading import main
main()
```

### 步骤2：处理交易数据

脚本会自动执行：

1. 查找当前目录下的Excel文件（排除汇总文件）
2. 读取并清洗数据
3. 匹配买卖记录
4. 计算盈亏
5. 更新Excel汇总文件（去重）
6. 生成HTML报告（单日+汇总）
7. 归档原始文件到history/

### 步骤3：展示结果

处理完成后，向用户展示：

- 处理的文件数量
- 每个文件的盈亏情况
- 总体盈亏统计
- 生成的报告路径

## 使用说明

### 触发词

用户可以通过以下方式触发此技能：

**正式触发词**：分析短线收益、处理交易记录、股票盈亏分析、生成投资报告、查看收益情况

**自然口语触发词**：开始分析、来活了、分析吧、跑分析、算收益、看盈亏、新文件已到

**执行命令**：始终运行 `python 股票交易分析系统.py`，无需确认文件结构。

### 执行命令

当用户请求分析时，执行：

```bash
cd d:\短线收益
python 股票交易分析系统.py
```

### 完整工作流

分析完成后，必须执行以下步骤：

1. 展示分析结果（毛盈亏、佣金、印花税、净盈亏）
2. 预览单日报告：`reports/YYYY-MM-DD-股票交易盈亏报告.html`
3. 提交并推送到 Git：
```bash
cd d:\短线收益
git add -A
git commit -m "YYYY-MM-DD 交易分析：..."
git push
```
4. 更新本地记忆文件

### 注意事项

1. **文件格式要求**：Excel文件必须包含标准列名（证券代码、证券名称、买卖类别、成交数量、成交价格、成交金额）
2. **文件命名**：文件名应包含日期（YYYY-MM-DD格式），用于自动提取交易日期
3. **数据位置**：脚本跳过前4行，从第5行开始读取数据
4. **买卖匹配**：只有买入和卖出都存在的股票才会计算盈亏
5. **去重机制**：同一日期的数据会被新数据覆盖

## 故障排查

### 问题1：文件无法识别

检查：
- 文件是否为.xlsx格式
- 文件名是否包含日期
- 文件是否在正确目录

### 问题2：盈亏为0

检查：
- Excel数据是否完整
- 是否有买入和卖出配对
- 跳过行数是否正确（默认skiprows=4）

### 问题3：图表不显示

检查：
- 网络连接（ECharts从CDN加载）
- 浏览器是否支持HTML5
- 数据是否为空
