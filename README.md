# 股票交易盈亏分析系统 使用说明

## 📋 系统简介

这是一个自动化的股票交易盈亏分析系统，可以自动处理券商导出的交易记录，计算盈亏并生成可视化报告。

## 🗂️ 文件结构

```
d:\短线收益\
├── 股票交易分析系统.py      # 主处理脚本
├── 股票交易盈亏汇总.xlsx    # Excel汇总数据
├── reports\                 # HTML报告存放文件夹
│   ├── YYYY-MM-DD-股票交易盈亏报告.html  # 单日报告
│   └── 汇总可视化报告.html               # 汇总可视化报告
├── history\                 # 原始文件归档文件夹
│   └── YYYY-MM-DD-两融-当日成交汇总.xlsx
└── README.md               # 本说明文件
```

## 🚀 使用方法

### 1. 准备文件
将券商导出的Excel交易记录文件放到 `d:\短线收益\` 根目录下

### 2. 运行脚本
```bash
python "股票交易分析系统.py"
```

### 3. 查看结果
- **Excel汇总**：`股票交易盈亏汇总.xlsx`（所有交易记录的累计数据，支持去重更新）
- **单日HTML报告**：`reports\日期-股票交易盈亏报告.html`（单日详细报告）
- **汇总可视化报告**：`reports\汇总可视化报告.html`（多维度图表分析）
- **原始文件**：已归档到 `history\` 文件夹

## 📊 功能特点

### 自动化处理
- ✅ 自动识别Excel文件
- ✅ 自动提取交易日期
- ✅ 自动匹配买卖记录
- ✅ 自动计算盈亏

### 智能计算
- 按股票代码配对买卖记录
- 买卖数量不一致时，取较小值计算
- 加权平均计算成交均价
- 自动跳过未配对记录

### 报告生成
- Excel汇总：支持去重更新，同日期数据自动覆盖
- 单日HTML报告：可视化展示单日交易详情
- 汇总可视化报告：多维度图表分析，包括：
  - 每日盈亏柱状图
  - 累计收益曲线
  - 各股票盈亏分析（柱状图+饼图）
- 中国股市配色：红涨绿跌

## 📈 计算逻辑

### 盈亏计算公式
```
匹配数量 = min(买入数量, 卖出数量)
买入均价 = 总买入金额 / 总买入数量
卖出均价 = 总卖出金额 / 总卖出数量
买入金额 = 买入均价 × 匹配数量
卖出金额 = 卖出均价 × 匹配数量
盈亏金额 = 卖出金额 - 买入金额
```

### 示例
```
股票：宁德时代 (300750)
买入：数量=200, 均价=444.0350, 金额=88807.00
卖出：数量=100, 均价=447.5600, 金额=44756.00
匹配：数量=100, 买入金额=44403.50, 卖出金额=44756.00
盈亏：+352.50元 (盈利)
```

## 💡 注意事项

1. **文件格式**：确保Excel文件包含标准列名（证券代码、证券名称、买卖类别、成交数量、成交价格、成交金额）
2. **命名规范**：文件名应包含日期（格式：YYYY-MM-DD），用于自动提取交易日期
3. **数据备份**：系统会自动归档原始文件，但建议定期备份 `股票交易盈亏汇总.xlsx`
4. **去重机制**：如果同一日期的数据已存在，系统会自动删除旧数据并重新生成
5. **可视化报告**：汇总可视化报告会在每次处理文件后自动更新，无需手动生成
